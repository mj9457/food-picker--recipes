계란볶음밥

🍚 재료(1인분 기준)

-밥 1공기
-계란 1~2개
-대파,버터,후추 (없어도 문제없음)
-굴소스 (진간장 또는 소금도 문제없음)

🍳 방법

1. 밥과 생계란을 비벼놓는다.
2. 팬이 차가울때 기름과 파를 같이 넣고 가열시켜 파기름을 낸다.
3. 파향이 맛있게 올라오면 비벼둔 밥을 넣고 같이 볶아준다.
4. 맛있게 먹는다.

주의사항

* 팬에 기름과 파를 같이 넣어 서서히 가열시키세요! *
* 굴소스 대신 간장이나 소금으로 간하세요! *
* 버터 안넣어도 괜찮습니다! *

식빵계란토스트

🐥 재료(1인분 기준)

-식빵 4개
-계란 3개
-설탕1스푼

🍳 방법

1. 계란과 설탕 1스푼을 섞어준다.

2. 팬에 식용유 넣고 마늘 먼저 볶다가 채소, 양념 넣고 볶기
3. 우동사리 넣고 2분후 불끄기
4. 후추, 마요네즈 등 뿌려 먹으면 더 맛있습니다

- 대파
- 소금

🍳 방법

2. 간 감자는 채에 받쳐 물기를 빼둔다 (약 10~15분)
3. 빠진 감자수분을 기울여서 윗물만 따라낸다
4. 바닥에 남은 전분과 간 감자를 섞는다 (대파, 소금 넣고 같이 섞기)
5. 팬에 기름을 적당히 두르고 부친다 (중불)

잔치국수

🍜 재료(1인분 기준)

- 국수(소면, 중면)
- 채소(양파, 애호박, 당근, 대파 등), 계란
- 육수용 멸치, 다시마
- 소금 또는 멸치액젓 또는 국간장

🍳 방법

양념장 : 양조간장1스푼 + 고춧가루 반스푼 + 다진마늘 반스푼 + 참기름 반스푼 + 깨 + 대파

1. 채소들은 가늘게 썰어 숨이 살짝 죽을 정도로 볶아둔다(소금살짝)
2. 계란은 잘 풀어서 약한불에 부쳐낸후 가늘게 썰어둔다
3. 냄비에 멸치 한줌, 다시마 2장을 넣어 물을 끓인다
4. 물이 끓으면 다시마만 건져내고, 약불로 줄인후 15분 더 끓인다
5. 15분후 멸치도 건져내고, 소금 또는 멸치액젓이나 국간장으로 간을 한다
6. 냄비에 물을 새로 끓이고 국수를 삶는다(물3번 넣고 4번째에 끄기)
7. 물기를 뺀 국수에 볶은 채소, 육수를 부어 양념장과 함께 먹는다

앙념 : 고추장3 / 고춧가루2 / 간장2 / 설탕1 / 마늘1 / 후추 톡톡
1. 양념을 만든 후 고기를 섞어둔다 (바로 볶아도 되고 냉장고에 30분 재워도 됨)
2. 팬에 기름을 살짝 두르고 고기를 넣고 볶는다 (중불)
3. 고기가 어느정도 익으면 양파를 넣고 볶는다
4. 모든재료가 다 볶아지면 대파를 넣고 휘릭 볶은후 불을 끈다

쿠앤크 아이스크림

🍫 재료(1인분 기준)

- 오레오 10~14개
- 우유 400ml
- 생크림 500g
- 설탕 4스푼 (취향껏 더 달게 해도 좋음)

🍳 방법

1. 우유에 설탕넣고 저어서 녹인후
2. 생크림 넣고 잘 섞고
3. 오레오도 잘 섞은후 얼린다(5시간 이상)

팽이버섯볶음

🍄 재료(1인분 기준)

- 팽이버섯 1봉
- 양파 / 마늘 / 대파 (그 외 좋아하는 채소)
- 소금 + 참기름

🍳 방법

1. 팽이 밑둥 자르고 절반으로 썬다 (안썰어도됨)
2. 채소는 잘고 얇게 썰어둔다
3. 팬에 식용유 + 마늘 노릇 볶는다
4. 마늘 노릇해지면 양파 넣어 살짝 볶는다
5. 팽이와 소금 넣고, 센불에 20초 볶는다 (!!오래 볶으면 흐물해짐!!)
6. 불 끄고 참기름 1, 대파 넣어 먹기

참치비빔밥

🍚 재료(1인분 기준)

- 참치캔 작은거
- 각종 채소(집에 있는거)
- 고추장 반스푼
- 물엿 반스푼
- 식초 반스푼
- 들기름

🍳 방법

양념장 : 고추장 반스푼 +물엿 반스푼 + 식초 반스푼 + 들기름
1. 각종 채소는 채를 썬다
2. 참치 기름은 따라 둔다
3. 밥 위에 채소와 참치를 얹어 먹는다

감자샌드위치

🍞 재료(1인분 기준)

- 빵 (아무 빵)
- 감자 2개, 계란 3개
- 마요네즈 + 머스타드
- 소금 + 설탕 약간
* !오이나 피클을 썰어 넣으면 더 맛이 좋음! *

🍳 방법

1. 감자는 토막내서 끓는물에 팍 익힌다
2. 계란도 끓는물에 12분 삶는다
3. 감자는 뜨거울 때 으깨고, 삶은 계란도 으깬다
4. 마요네즈, 머스타드, 소금, 설탕을 입에 맞을때까지 추가한다
5. 빵 사이에 넣어 먹거나 그냥 먹는다

로제 라볶이

🍜 재료(1인분 기준)

- 까르보불닭 1봉
- 떡, 어묵, 소세지
- 케첩1
- 설탕1 (달짝지근하게 먹으려면 2)
- 고추장 반스푼
*취향껏 고춧가루도 넣으면 좋음*

🍳 방법

1. 끓는물에 면 넣고 풀어지기만 하면 호다닥 건져둔다
2. 팬에 우유 두잔 정도 붓고 라면 소스 + 위 양념을 넣는다
3. 끓으면 면만 빼고 모든 재료를 다 넣고 끓인다. 약불
4. 적당히 꾸덕의 기운이 느껴지면 면을 넣고 마저 익히면 끝!
*취향껏 고춧가루, 슬라이스 치즈, 후추 등을 넣어 먹는다*

참치김치찌개

🍜 재료(1인분 기준)

- 김치 한 공기
- 참치 1캔
- 양파 + 대파
- 설탕 반스푼 + 고춧가루 반스푼

🍳 방법

1. 참치기름을 냄비에 붓고 김치와 볶기 (설탕 반스푼 넣기)
2. 김치가 적당히 볶아지면 양파 넣고 볶기
3. 양파가 투명해지면 물 1컵 붓기
4. 중약불에서 끓이다가 5분 뒤 고춧가루 반스푼, 참치 넣기
5. 참치는 젓지말고 가만히 둔채로 5분 더 끓인다
6. 대파 넣고 한번 부르르 끓으면 불 끄기
